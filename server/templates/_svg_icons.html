{# _svg_icons.html — single shared inline-SVG sprite for the entire SLink web UI. Included once per page via base.html so every server.py / manager.py / stream template can reference symbols with ```` and have them pick up the active theme's `--c-*` tokens via `fill: currentColor`. Design notes: - 16×16 viewBox throughout; matches the dashboard sidebar's nav-ico grid and the manager's mgr-btn-ico sizing. - Single-path glyphs wherever possible (no nested g/groups) so each symbol stays under ~200 bytes. - Pixel-art style (right-angle fills) matches the Pixelify Sans body voice rather than the smooth-vector look of Material/Phosphor. - `fill="currentColor"` everywhere so wrapping text colour drives the icon hue. Themes that want a per-icon accent (e.g. gold for shiny) set `color: var(--c-gold)` on the parent span. When adding a new icon: keep `viewBox="0 0 16 16"`, one ``, and describe it briefly above the line so the next person knows what it represents. #}
