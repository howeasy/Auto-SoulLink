# FRLG Game Data

Data files for Gen 3 FireRed/LeafGreen (vanilla, Archipelago, Radical Red).

## Files

- `area_map.json` — 175-entry area ID lookup table
- `rom_map_names.json` — ROM map name data
- `rom_mapsec_names.json` — ROM map section names
- `rr_items.json` — 746 Radical Red item ID → name mappings
- `rr_species.json` — RR species data
- `rr_sprites.json` — RR sprite filename mappings
- `rr_trainers.json` — RR trainer data
- `rr_types.json` — 1328 RR species type overrides
